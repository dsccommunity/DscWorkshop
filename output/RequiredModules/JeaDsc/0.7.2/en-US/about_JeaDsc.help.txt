SHORT DESCRIPTION
    DSC resource to create and control Just Enough Administration endpoints and role capability files with DSC.

LONG DESCRIPTION
    DSC resource to create and control Just Enough Administration endpoints and role capability files with DSC. This module
    contains a resource to create the role capability files (.psd1) and another one for creating endpoints which make use of
    the roles.

EXAMPLES
    PS C:\> Get-DscResource -Module JeaDsc
